根据乙方在甲方平台下达的订单内容显示，双方确认本次二手手机交易的标的手机为乙方自有的、容量G的手机一台，手机回收价XXX元。甲乙双方为了维护双方利益，安全达成本次二手手机交易，甲乙双方在平等、自愿、公平、诚实、信用的基础上，根据相关法律、法规的规定，就旧手机回收有关事宜，达成协议如下：

第一章 甲方、乙方权利与义务

1、 甲方认同并遵守以下条款

（1）甲方应当为乙方提供专业化的、更优质的回收和评估服务；

（2）甲方应当为乙方的个人信息进行必要保护；

（3）甲方应当按照约定支付回收款；

（4）甲方应当在收到手机应当及时完成检测，并将检测结果通知乙方。

2、 乙方认同并遵守以下条款：

（1）乙方在交易过程中应当为甲方提供真实、详尽及准确的个人资料。

（2）乙方所有向甲方提供的资料将视为乙方同意向甲方公开信息。

（3）乙方在享用甲方各项服务的同时，视为同意并接受本服务协议。

（4）乙方确保出售给甲方的旧手机为乙方合法购买或拥有的品牌产品，非山寨机、仿冒机或贴牌机或通过非正规渠道、以非法方式获得。

（5）乙方需如实填选手机的真实情况，不得恶意操作。恶意操作指手机的实际情况与选择项严重不符，包括以下几种情况：

①手机型号不符； ②手机的帐号密码未解除； ③手机屏幕未达到《手机折旧程度对照表》相应标准； ④手机外壳未达到《手机折旧程度对照表》相应标准； ⑤手机无法正常使用各项功能； ⑥手机拆修或进水； ⑦手机硬件有损坏； ⑧手机有网络锁； ⑨其他与乙方订单选填内容不符的情况。

第二章 回收价格

1、 在平等、自原、公平、诚实、信用的基础上，甲方严格按照专业、准确的产品评定及定价系统来核算产品对应回收的价格。

2、 乙方询价后勾选本协议并提交订单后，视为乙方认同并接受甲方作出的价格判定方法和结果。

第三章 付款、送达及检测

1、 甲方在乙方下单后，会根据乙方提供的信用分，按照回收预估价格，先向乙方指定的银行帐户内汇入手机回收款。在甲方付款后，乙方应按照协议约定交货，交货的时间、方式、质量等应严格按照订单及本协议要求，否则需支付xxx元违约金。

2、乙方如进行履约应当在收到甲方支付的手机回收款之日起，十四日内向甲方邮寄送达手机。无论何种原因造成超时送达的，甲方均有权选择拒收，并要求乙方承担违约责任。送达前甲方不承担货物灭失、损毁的风险。

3、乙方除应交付手机外，还应将手机质量合格凭证、购买发票、单据、手机官方配件等配套资料一并交付，否则视为乙方未交付。

4、甲方作为专业的手机回收商，乙方认同甲方的检测和判定规则，数据均以甲方检测结果为准，乙方享有知情权。甲方收到乙方的物品后，会进行不损害物品的必要检验，如检验结果与乙方订单中描述一致，则本次交易完成；如检验结果与描述不一致，则甲方有权解除本回收协议，并要求乙方承担违约责任。

第四章 违约责任

1、双方认同并理解本协议所约定的事项均为本次二手手机回收交易的基础。乙方如违反本协议部分或全部的内容，则均视为根本违约。甲方有权单方解除本协议，取消本次回收交易，并将产品退回乙方。乙方应当在甲方解除本协议之日起14日内立即退还甲方已支付的手机回收款，同时向甲方支付违约金XXX元。

2、乙方没有按照约定立即退还货款并支付违约金的，甲方有权要求乙方额外支付逾期履行的费用，每日收取交易金额与违约金之和的2%作为滞纳金。 第五章 免责条款

3、乙方确认在交易前，手机中的所有用户个人信息、资料、图片、通讯录、视频、音频、应用软件等（统称“用户信息”）均已由乙方自行采取备份、删除等手段处理。甲方付款后，即视为乙方放弃用户信息的保密、备份要求，甲方有权通过锁定手机等方式，清除用户信息，对此行为及由此可能造成的损失，甲方无需承担任何责任。

4、不论在任何情况下，甲方均不对由于互联网设备正常维护、互联网络连接故障，设备终端、通讯或其他系统的故障，电力故障，罢工、暴乱、骚乱、灾难性天气（如火灾、洪水、风暴等)、爆炸、战争，政府行为，司法机关的命令或第三方不作为而造成的不能履行或延迟履行承担责任。

5、因数据错误等原因造成回收预估价格明显不合理的（如回收预估价格超过新品价格或其他依据常识不可能的价格），乙方同意甲方有权校对此价格进行修正而无需向乙方承担违约责任。

第六章 法律纠纷

1、本协议条款中的任何一项与中华人民共和国现行法律的强制性规定相悖时，以法律的强制性规定为适用依据，除此之外，不违背法律强制性规定的条款，继续有效。

2、双方一致确认本协议中双方的联系方式真实有效，并且同意仲裁、诉讼、执行等程序由以上联系方式向乙方（短信送达、邮箱送达、邮寄送达、留置送达、公告送达等任意选择）送达文书和通知，协议中约定的送达地址一经投递视为送达。

3、甲、乙双方一致同意，因纠纷产生的诉讼费、鉴定费、律师费、公告费、公证费等费用由违约方承担。
